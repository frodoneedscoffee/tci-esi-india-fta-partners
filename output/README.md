# Output Folder

This folder contains generated outputs from the empirical analysis.

## Contents

### 1. Index Datasets
- Year-wise Trade Complementarity Index (TCI)
- Year-wise Export Similarity Index (ESI)
- Calculated at:
  - HS2 (sector-level)
  - HS4 (product-level)

These CSV files contain:
- Year
- Partner
- Index value

### 2. Product-Level Files
- Absolute differences in trade shares (used to compute TCI)
- Product-level complementarity measures

### 3. Visualizations
- Line plots (time trends)
- Quadrant classification graphs (TCI vs ESI)

All plots are exported as PNG files.

## Interpretation Guide

- High TCI → India's export structure matches partner's import demand.
- High ESI → India and partner export similar baskets (competition/intra-industry potential).
- HS2 → Sector-level structure.
- HS4 → Product-level specialization.

These outputs are generated by scripts located in the `scripts/` folder.
